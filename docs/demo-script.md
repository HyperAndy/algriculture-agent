# 大田作物稳产减灾 Agent MVP 3分钟演示脚本

1. 打开 `http://localhost:3000/demo`，介绍项目面向水稻、小麦、玉米、大豆，解决农技覆盖不足、风险发现晚、水肥用药粗放和农事档案分散的问题。
2. 展示大屏中的核心指标、作物分布和六 Agent 流程，强调系统不是单轮问答，而是“多源感知 → 作物诊断 → 风险预警 → 水肥决策 → 灾害应对 → 农事档案”的闭环。
3. 进入 `http://localhost:3000/console`，说明这是农技员和合作社日常使用的管理后台。
4. 打开地块管理，选择“东北玉米示范田”。
5. 进入观测录入页，使用默认的高温少雨、墒情偏低、叶色偏淡和虫害描述样例，保存并进入 Agent 分析。
6. 点击“开始Agent分析”，展示生成的综合风险、长链推理、农事建议和六个 Agent 步骤。
7. 打开任务页，展示系统自动生成“安排补灌”“病虫草害复查”“评估追肥方案”和“复查地块状态”等任务。
8. 打开档案页，说明每次分析都会沉淀为可追溯记录，可支撑项目验收、生产追溯、补贴申请和保险理赔。
