# 大田作物稳产减灾 Agent 风险评分与农事建议规则

## 1. 文档目的

本文档定义 MVP 阶段风险评分、综合风险合并、农事建议和任务生成规则，确保 Agent 输出稳定、可解释、可测试。

## 2. 风险等级

| 等级 | 值 | 说明 |
|---|---|---|
| 低风险 | `low` | 当前数据未显示明显异常，保持常规巡田 |
| 中风险 | `medium` | 存在异常信号，需要加强观察或采取轻量措施 |
| 高风险 | `high` | 存在较强风险信号，需要尽快安排农事处置 |

综合风险取各专项风险中的最高等级。

## 3. 干旱风险

| 条件 | 风险 |
|---|---|
| 墒情 `< 40%` 且温度 `>= 32℃` | 高 |
| 墒情 `< 45%` 且天气趋势包含“高温少雨” | 高 |
| 墒情 `< 50%` | 中 |
| 天气趋势包含“少雨”或“持续高温” | 中 |
| 其他 | 低 |

建议：

- 高风险：24 小时内安排补灌，避开高温时段追肥。
- 中风险：加强墒情监测，结合天气窗口安排灌溉。
- 低风险：保持常规巡田。

## 4. 涝渍风险

| 条件 | 风险 |
|---|---|
| 降雨量 `>= 80mm` 或天气趋势包含“暴雨”“强降雨” | 高 |
| 降雨量 `>= 30mm` 或天气趋势包含“连续降雨” | 中 |
| 其他 | 低 |

建议：

- 高风险：检查排水沟渠，提前疏通低洼地块。
- 中风险：关注田间积水，雨后巡田。
- 低风险：无需特殊处置。

## 5. 高温风险

| 条件 | 风险 |
|---|---|
| 温度 `>= 35℃` | 高 |
| 温度 `>= 32℃` | 中 |
| 其他 | 低 |

作物特殊规则：

- 小麦灌浆期 + 高温少雨：提示干热风风险。
- 玉米抽雄吐丝期 + 高温少雨：提示授粉受影响风险。
- 大豆开花期 + 高温少雨：提示落花落荚风险。
- 水稻抽穗扬花期 + 高温：提示结实率风险。

## 6. 长势风险

| 条件 | 风险 |
|---|---|
| `leafColor = pale` 且 `growthStatus = weak` | 高 |
| `leafColor = pale` 或 `growthStatus = weak` | 中 |
| `growthStatus = vigorous` 且无其他异常 | 低 |
| 其他 | 低 |

建议：

- 高风险：复查墒情、根系、病害和养分，评估追肥或灌溉。
- 中风险：3 天内复查长势，关注叶色和株高变化。

## 7. 养分风险

| 条件 | 风险 |
|---|---|
| 氮磷钾中两个及以上为 `low` | 高 |
| 氮磷钾任一为 `low` | 中 |
| 氮磷钾均为 `normal` | 低 |
| 任一为 `high` | 中，提示避免过量投入 |

建议：

- 缺素风险：评估追肥方案，避免高温时段施肥。
- 过量风险：减少重复施肥，关注倒伏和环境风险。

## 8. 病虫草害风险

| 条件 | 风险 |
|---|---|
| 描述包含“严重”“大量”“扩散”“病斑”“虫口”“成片” | 高 |
| 任一病虫草害描述不为空 | 中 |
| 全部为空 | 低 |

建议：

- 高风险：尽快巡田复核，必要时联系农技员确认防治方案。
- 中风险：2 天内复查发生范围和发展趋势。
- 低风险：常规巡田。

## 9. 作物专项规则

### 9.1 水稻

- 分蘖期 + 墒情偏低：提示分蘖不足风险。
- 抽穗扬花期 + 高温：提示结实率风险。
- 连续降雨 + 高湿描述：提示病害风险升高。

### 9.2 小麦

- 灌浆期 + 高温少雨：提示干热风风险。
- 抽穗期 + 连续降雨：提示赤霉病风险升高。
- 拔节期 + 氮素高：提示倒伏风险。

### 9.3 玉米

- 拔节期 + 墒情低：提示拔节生长受限风险。
- 抽雄吐丝期 + 高温少雨：提示授粉风险。
- 大喇叭口期 + 虫害描述：提示加强虫害巡查。

### 9.4 大豆

- 开花期 + 高温少雨：提示落花落荚风险。
- 结荚期 + 墒情低：提示结荚和鼓粒受限风险。
- 病斑描述：提示叶部病害巡查。

## 10. 综合风险合并

风险等级映射：

```text
low = 1
medium = 2
high = 3
```

综合风险 = 干旱、涝渍、高温、长势、养分、病虫草害、作物专项风险中的最高等级。

## 11. 建议优先级

| 风险 | 任务优先级 | 截止时间 |
|---|---|---|
| 高风险干旱 | high | 24 小时内 |
| 高风险病虫草害 | high | 24-48 小时内 |
| 中风险病虫草害 | medium | 2 天内 |
| 中风险养分 | medium | 3 天内 |
| 固定复查 | low/medium | 3 天后 |

## 12. 任务生成规则

每次分析至少生成一条“复查地块状态”任务。

触发任务：

- 干旱高风险：生成“安排补灌”。
- 涝渍高风险：生成“检查排水沟渠”。
- 病虫草害中高风险：生成“开展病虫草害复查”。
- 养分中高风险：生成“评估追肥方案”。
- 长势中高风险：生成“复查苗情长势”。

## 13. 输出解释要求

每条高风险结论必须包含：

- 风险对象：例如干旱、病虫害、干热风。
- 触发依据：例如墒情 38%、温度 34℃、未来 5 天高温少雨。
- 建议措施：例如补灌、巡田、排水、复查。
- 时间窗口：例如 24 小时内、2 天内、3 天后复查。
