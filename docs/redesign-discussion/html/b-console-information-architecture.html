业务模块边界

新建观测

1. 工作台

全局态势与今日工作入口。

  • 核心指标
  • 高风险地块
  • 待办任务
  • 近期报告

2. 地块管理

农田生产对象主数据。

  • 地块列表
  • 地块详情
  • 作物与品种
  • 风险状态

3. 观测录入

触发分析前的数据采集。

  • 天气
  • 土壤
  • 苗情
  • 病虫草害

4. Agent 分析

多 Agent 协作与推理透明化。

  • 启动分析
  • 步骤时间线
  • 风险判断
  • 建议生成

5. 农事任务

把建议转成可执行任务。

  • 任务队列
  • 优先级
  • 状态流转
  • 批量处理

6. 报告归档

验收、复盘和追溯材料。

  • 报告列表
  • 报告详情
  • 导出
  • 历史检索
